# API Agência de Viagens

Projeto desenvolvido para a disciplina de Desenvolvimento de Sistemas Web.

## Tecnologias Utilizadas

* Java 17
* Spring Boot
* Spring Data JPA
* Spring Security
* PostgreSQL
* Maven

## Funcionalidades

* Cadastrar destinos de viagem
* Listar destinos
* Pesquisar destinos
* Consultar destino por ID
* Avaliar destinos
* Atualizar destinos
* Excluir destinos

## Perfis de Usuário

### ADMIN

Pode acessar todos os endpoints da API.

### USER

Pode consultar e avaliar destinos.
